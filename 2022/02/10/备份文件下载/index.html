网站源码

当开发人员在线上环境中对源代码进行了备份操作，并且将备份文件放在了 web 目录下，就会引起网站源码泄露。

常见的网站源码备份文件名：

  • web
  • website
  • backup
  • back
  • www
  • wwwroot
  • temp

常见的网站源码备份文件后缀：

  • tar
  • tar.gz
  • zip
  • rar
